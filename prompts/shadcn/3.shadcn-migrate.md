# shadcn/ui 移行プロンプト

## 目的

既存ハイファイUIを
Design Tokenに準拠したshadcnコンポーネントへ置き換える。

## 置換ルール

- button → shadcn Button
- modal → Dialog
- card → Card
- input → Input
- textarea → Textarea
- badge → Badge

## 制約

- ルーティング構造は変更しない
- コンポーネント単位で段階置換
- デザインは既存トークンに準拠
- API実装は禁止

## 出力

- 変更ファイルのみ
- フルコード
- 説明禁止
