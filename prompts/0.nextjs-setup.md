# Next.js 16 App Router Project Setup Prompt

以下の要件に従い、プロダクション品質のNext.jsプロジェクトをセットアップしてください。

---

## ■ 基本要件

- Next.js 16
- App Router 構成
- TypeScript latest
- Tailwind CSS latest
- ESLint
- Prettier
- 状態管理ライブラリは使用しない（React標準のみ）
- src/ ディレクトリ構成を採用
- パッケージマネージャは npm

---

## ■ デプロイ要件

- GitHub Pages にデプロイ可能な完全静的サイト構成
- `output: "export"` を設定
- `next build` → `next export` が成功すること
- 画像最適化は無効化する

### next.config.ts に含める設定

- output: "export"
- images.unoptimized: true
- trailingSlash: true

※ basePath は未指定（ユーザーサイト前提）

---

## ■ ディレクトリ構成

```
src/
├── app/                      # Next.js App Router
│   ├── layout.tsx            # ルートレイアウト
│   ├── page.tsx              # ホームページ
│   ├── loading.tsx           # ローディングUI
│   ├── error.tsx             # エラーUI
│   ├── not-found.tsx         # 404ページ
│   ├── [dynamic]/            # 動的ルート
│   │   └── page.tsx
│   └── api/                  # API Routes
│       └── [endpoint]/
│           └── route.ts
├── components/               # Reactコンポーネント
│   ├── ui/                   # 汎用UIコンポーネント（Button, Card等）
│   ├── features/             # 機能単位のコンポーネント（StoryList, CommentSection等）
│   └── layouts/              # レイアウトコンポーネント（Header,Footer等）
```

## ■ ESLint 設定

- next/core-web-vitals をベース
- unused vars は warning
- import順の自動整列は不要

---

## ■ Prettier 設定

- semi: true
- singleQuote: true
- trailingComma: "all"
- printWidth: 100
- tabWidth: 2

---

## ■ Tailwind 設定

- 最新安定版
- darkMode: "class"
- content は src 配下を対象に設定

## ■ package.json scripts

以下を含めること:

```
  "scripts": {
    "dev": "next dev -p 3000",
    "build": "next build",
    "start": "next start",
    "lint": "next lint",
    "format": "prettier --write ."
  },
```

## ■ 追加要件

- TypeScript strict: true
- 不要な依存は含めない
- コメントは最小限
- 初期ページはシンプルなTailwindスタイルのトップページ
- レイアウトは <html lang="ja"> を設定
- メタデータを適切に設定
