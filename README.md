# ui-wireframe-template

このリポジトリは、UI ワイヤーフレームのテンプレートプロジェクトです。

目的
- シンプルなワイヤーフレーム構成を提供し、プロトタイピングやデザイン確認に利用できる雛形を提供します。

主な内容
- src/ : フロントエンドのソースコード（コンポーネントやスタイル）
- prompts/ : プロジェクトに関連するプロンプトやメモ

クイックスタート
1. リポジトリをクローン
   git clone <このリポジトリのURL>
2. 依存関係がある場合はインストール（プロジェクトに package.json がある場合）
   npm install
3. 開発サーバーを起動（スクリプトが定義されている場合）
   npm run dev
